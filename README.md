# Tampermonkey Scripts

Personal Tampermonkey userscripts.

## Layout

- `scripts/`: Tampermonkey `.user.js` scripts.
- `test/`: Node tests for pure script logic.

## Scripts

- `scripts/m-team-torrent-enhancer.user.js`: Enhances M-Team torrent lists with new-hot highlighting and viewed-row dimming.

## Notes

- Keep script names in `kebab-case.user.js`.
- Avoid committing site tokens, cookies, or other private data.
